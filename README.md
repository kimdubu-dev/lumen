# 🌙 Lumen

Lumen은 Flutter와 Firebase를 활용하여 개발한 감정 기반 소셜 다이어리 애플리케이션입니다.

사용자는 하루의 감정을 기록하고, 친구들과 일기를 공유하며, 좋아요와 댓글을 통해 소통할 수 있습니다.

---

## ✨ 주요 기능

### 🔐 사용자 인증

* 회원가입
* 로그인
* 자동 로그인
* 로그아웃

### 📖 다이어리

* 일기 작성
* 일기 수정
* 일기 삭제
* 캘린더 기반 일기 관리
* 감정 이모지 선택
* 공개 / 비공개 설정

### 👤 프로필

* 프로필 사진 변경
* 닉네임 변경

### 👥 친구 시스템

* 친구 검색
* 친구 요청
* 친구 수락
* 친구 목록 조회

### 🌐 소셜 기능

* 친구 공개 일기 피드
* 좋아요 기능
* 댓글 기능

## 🚀 실행 방법

### 프로젝트 클론

```bash
git clone https://github.com/kimdubu-dev/lumen.git
cd lumen
```

### 패키지 설치

```bash
flutter pub get
```

### 실행

```bash
flutter run
```

### APK 빌드

```bash
flutter build apk --release
```
